## 🎯 게임 개요

라이어게임은 심리전과 추론이 핵심인 파티게임입니다. 플레이어들은 주어진 단어에 대해 설명하면서 라이어를 찾아내거나, 라이어는 정체를 숨기고 단어를 맞춰야 합니다.

## ✨ 주요 특징

- 🔥 **완전 휘발성 설계** - 호스트 퇴장 시 즉시 방 삭제
- ⚡ **실시간 동기화** - Firebase Realtime Database로 즉시 반영
- 🎮 **URL 기반 방 관리** - 6자리 코드로 간편한 방 공유
- 📱 **모바일 최적화** - QR코드로 쉽게 참여
- 🎨 **글래스모피즘 UI** - 모던하고 세련된 디자인
- 🔄 **자동 복구** - 새로고침해도 게임 상태 유지
- 🧹 **자동 정리** - 24시간 후 자동 방 삭제

## 🎮 게임 규칙

### 📋 기본 게임 플레이
- **플레이어 수**: 3명 이상 (최대 10명)
- **게임 시간**: 10-15분
- **목표**: 라이어를 찾아내거나, 라이어는 정체를 숨기고 단어를 맞춰야 함

### 🎭 역할 분배

#### 3-5명
- **일반인**: (플레이어 수 - 1)명
- **라이어**: 1명

#### 6-7명
- **일반인**: 4-5명
- **광신도**: 1명
- **라이어**: 1명

#### 8명 이상
- **일반인**: 5-7명
- **광신도**: 1명
- **라이어**: 2명

### 📢 정보 공유 방식

#### 👤 일반인 & 🔥 광신도
- **주제 + 단어** 공유
- 예: *"주제는 '음식'이며 단어는 '떡볶이'입니다."*

#### 🎭 라이어
- **주제만** 공유
- 예: *"주제는 '음식'입니다. 당신은 라이어이며 단어를 유추하여 맞춰야 합니다."*

### 🔄 게임 진행 순서

1. **🎯 주제 선택**: 호스트가 게임 주제 선택
2. **🎭 역할 배정**: 플레이어들에게 역할 랜덤 배정
3. **✅ 역할 확인**: 모든 플레이어가 자신의 역할 확인
4. **💬 설명 단계**: 돌아가면서 제시어에 대한 설명 발언
5. **🗳️ 투표 단계**: 라이어로 의심되는 사람 비밀 투표
6. **📢 최후 발언**: 가장 많이 지목된 사람이 최후의 발언
7. **🔄 철회 단계**: 투표 변경 기회 (철회/철회 안함 선택)
8. **🏆 결과 공개**: 역할 공개 및 승리자 결정

### 🏆 승리 조건

#### 🎭 라이어 승리
- **일반인이 라이어로 지목당함** OR
- **본인이 지목당했을 때 제시어를 정확히 맞힘**

#### 👤 일반인 승리
- **라이어를 찾아야함. 그리고 그 라이어가 제시어를 맞히지 못해야함**

#### 🔥 광신도 승리 (6명 이상)
- **본인이 라이어로 지목당함**

### 🎯 특별 규칙

- **비밀 투표**: 누가 누구에게 투표했는지 공개되지 않음
- **철회 시스템**: 투표 후 1회 변경 가능 (명확한 선택)
- **최후 발언**: 지목된 사람은 자신을 변호할 기회 제공
- **단어 맞히기**: 라이어가 지목되면 정확한 단어를 맞혀야 생존
- **자기 자신 투표 방지**: 자신을 지목할 수 없음

## 🚀 시작하기

### 1. 방 만들기
- 첫 번째 접속자가 자동으로 호스트가 됩니다
- 닉네임을 입력하고 "방 만들기" 클릭
- 6자리 방 코드가 자동 생성됩니다

### 2. 방 공유하기
- "공유" 버튼으로 링크 복사 또는 QR코드 생성
- 다른 플레이어들에게 링크 공유

### 3. 방 참여하기
- 링크로 접속한 플레이어들이 닉네임 입력
- "방 참여하기" 클릭하여 참여

### 4. 게임 시작
- 3명 이상 모이면 호스트가 "게임 시작" 클릭
- 자동으로 주제 선택 및 역할 배정

- **비밀 투표**: 익명성 보장
- **철회 시스템**: 명확한 선택 (철회/철회 안함)
